# 💰 Budget Tracker

A monthly budget tracking app with charts built with React and Vite.

## Live Demo
https://kenny-harper-budget-tracker.netlify.app

## Features
- Add income and expenses with categories
- Monthly breakdown with month navigation
- Balance, income and expense summary cards
- Pie chart showing spending by category
- Bar chart showing income vs expenses
- Data persists with localStorage

## Tech Stack
- React
- Vite
- Recharts (charting library)
- CSS (custom styling)

## Concepts Used
- useReducer for centralised state management
- useMemo for cached calculations
- useEffect for localStorage sync
- Props and component structure
- Array methods (.map, .filter, .reduce)
- Data visualisation with Recharts
